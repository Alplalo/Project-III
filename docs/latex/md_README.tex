 \hypertarget{md_README_autotoc_md1}{}\doxysection{\texorpdfstring{Brief description of the project}{Brief description of the project}}\label{md_README_autotoc_md1}
This project involves the development of a Molecular Dynamics code to simulate a Van der Waals gas, performing calculations in both serial and parallel modes.

Additionally, this project requires collaborative work to ensure the proper functioning of the code.\hypertarget{md_README_autotoc_md2}{}\doxysection{\texorpdfstring{Team Members and Responsibilities}{Team Members and Responsibilities}}\label{md_README_autotoc_md2}
Each team member has assigned tasks, which are indicated below\+:


\begin{DoxyEnumerate}
\item {\bfseries{Anna Monclús (@anna-\/mr98)}}\+: Initialize the configuration and define boundary conditions. Also coordinates the Git\+Hub repository.
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item {\bfseries{Aina Gaya (@ainagaya)}}\+: Integration Newton\textquotesingle{}s equations.
\item {\bfseries{Albert Plazas (@\+Alplalo)}}\+: Compute the forces for a Van der Waals interaction.
\item {\bfseries{Manel Serrano (@gluoon8)}}\+: Post-\/processing of data, statistics, and visualization.
\end{DoxyEnumerate}\hypertarget{md_README_autotoc_md3}{}\doxysection{\texorpdfstring{Prerequisites}{Prerequisites}}\label{md_README_autotoc_md3}
To execute the program, there are some pre-\/requisites\+:
\begin{DoxyItemize}
\item Make\+: to execute the program (\href{https://www.gnu.org/software/make/\#download}{\texttt{ https\+://www.\+gnu.\+org/software/make/\#download}})
\item Gfortran\+: to run the MD simulation. (\href{https://gcc.gnu.org/wiki/GFortran}{\texttt{ https\+://gcc.\+gnu.\+org/wiki/\+GFortran}})
\item Python 3.\+x \+: to generate the plots after simulation
\begin{DoxyItemize}
\item Numpy (\href{https://numpy.org/install/}{\texttt{ https\+://numpy.\+org/install/}})
\item Matplotlib (\href{https://matplotlib.org/stable/users/installing/index.html}{\texttt{ https\+://matplotlib.\+org/stable/users/installing/index.\+html}})
\end{DoxyItemize}
\end{DoxyItemize}\hypertarget{md_README_autotoc_md4}{}\doxysection{\texorpdfstring{How to}{How to}}\label{md_README_autotoc_md4}
\begin{quote}
\mbox{[}!\+IMPORTANT\mbox{]} Current features are only available for serial code, parallel code is WIP! \end{quote}

\begin{DoxyEnumerate}
\item Clone repository to your local host
\item Choose {\itshape serial} or {\itshape parallel (WIP)} folder with {\ttfamily cd serial} or {\ttfamily cd parallel}
\item Use {\ttfamily make} or {\ttfamily make help} to see the available commands.
\item Before starting a simulation, change your parameters in {\itshape nam\+MD.\+nml} file ~\newline

\item To carry out the simulation, use {\ttfamily make run} and the program will be compiled and run.
\item Data is generated in \texorpdfstring{$\ast$}{*}.dat\texorpdfstring{$\ast$}{*} files. If you want to generate figures, use {\ttfamily make plot}
\end{DoxyEnumerate}\hypertarget{md_README_autotoc_md5}{}\doxysubsection{\texorpdfstring{Quick guide}{Quick guide}}\label{md_README_autotoc_md5}
To carry out a simulation after choosing parameters in {\itshape nam\+MD.\+nml} file you can use\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{make\ run}
\DoxyCodeLine{make\ plot}

\end{DoxyCode}
 And files will appear in your main directory!

\begin{quote}
\mbox{[}!\+TIP\mbox{]} There are some ways to clean generated files, have a look at {\ttfamily make clean}, {\ttfamily make cleandata} and {\ttfamily make cleanplot}. \end{quote}
